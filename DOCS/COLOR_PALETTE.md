# Color Palette: macOS Luxury 🎨

Standardized colors used for buttons, labels, and surfaces.

## 1. System Accents
| Name | Hex | Usage |
| :--- | :--- | :--- |
| **System Blue** | `#007AFF` | Primary Buttons, Active States |
| **System Green** | `#34C759` | Success Icons |
| **System Red** | `#FF3B30` | Danger, Delete |
| **System Orange** | `#FF9500` | Warnings |

## 2. Surfaces (Dark Mode)
| Name | Hex/RGBA | Usage |
| :--- | :--- | :--- |
| **Secondary BG** | `#1E1E1E` | Code Window Body |
| **Glass BG** | `rgba(28, 28, 30, 0.7)` | Toolbar, Sidebar |
| **Border White** | `rgba(255, 255, 255, 0.1)`| Bevels, Dividers |

## 3. Typography
| Name | Hex/RGBA | Usage |
| :--- | :--- | :--- |
| **Primary Label** | `#FFFFFF` | Main Text |
| **Secondary Label** | `rgba(235, 235, 245, 0.6)`| Subtitles, Neutral Text |
| **Tertiary Label** | `rgba(235, 235, 245, 0.3)`| Placeholders, Disabled |

## 4. Backdrop Presets (Sample Gradients)
- **Cotton Candy**: `linear-gradient(135deg, #FF9A9E 0%, #FAD0C4 100%)`
- **Deep Space**: `linear-gradient(135deg, #1e3c72 0%, #2a5298 100%)`
- **Apple Aurora**: `radial-gradient(at 0% 0%, #764BA2 0%, transparent 50%), radial-gradient(at 100% 0%, #667EEA 0%, transparent 50%)`
